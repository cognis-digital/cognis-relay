# Limitations & Responsible Use

- **Planning/assessment tool, not a radio.** Cognis Relay validates PACE plan
  *structure* and simulates failover logic; it does not transmit, and it does not
  compute RF link budgets from first principles.
- **LPI/LPD posture is an operator-supplied score**, not a measured emission
  characteristic. Treat resilience scores as relative planning aids.
- **Failover model is discrete and priority-based** (P>A>C>E) over supplied
  outage states; real degradation is continuous and partial.
- **Sample plans/outages are illustrative.** Operational use requires validated
  platform, waveform, and link-budget data.

Use only within your lawful authority (LICENSE §9, NOTICE).
